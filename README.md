# Guide: How to Install APA 7th Edition in Microsoft Word
# by Kristian Daae-Johanse / IamPhasm 
Microsoft Word often comes pre-installed with APA 6th Edition, but most universities and publishers now require **APA 7th Edition**. Since Microsoft doesn't always provide an automatic update for this, you can install it manually following these steps.

---

## 1. Download the Style File
First, you need the `APASeventhEdition.xsl` file that contains the formatting rules for APA 7.

* **Download:** [APASeventhEdition.xsl @ IamPhasm Github](https://github.com/misterlisty/APA-7th-Edition-Style-for-Microsoft-Word)).
* *Note: If the link opens a page of code, right-click and select "Save As" to save it as a file.*

---

## 2. Locate the Bibliography Folder
You **must** place the downloaded file into Word's specific "Style" folder. The path depends on your operating system:

### For Windows:
1. Press `Windows Key + R` on your keyboard.
2. Copy and paste the following path and press **Enter**:
   `%AppData%\Microsoft\Bibliography\Style`
3. Paste the `APA7.xsl` file into this folder.

### For Mac:
1. Open **Finder**.
2. Click **Go** in the top menu bar, hold down the `Option` (Alt) key, and select **Library**.
3. Navigate to: 
   `Group Containers > UBF8T346G9.Office > User Content > Citation`
4. Paste the `APA7.xsl` file here.

---

## 3. Activate APA 7 in Word
1. **Restart Microsoft Word** (if it was already open).
2. Open your document.
3. Go to the **References** tab in the top ribbon.
4. In the **Style** dropdown menu, you should now see and be able to select **APA 7th Edition**.

---

## Pro Tips

* **Restart is Key:** If the style doesn't appear immediately, ensure the file is not inside a subfolder and that Word has been fully closed and reopened.
* **Language Settings:** Word may automatically translate terms like "and" or "editor" based on your document's proofing language.
* **Advanced Tools:** For large projects like a thesis or dissertation, consider using dedicated reference managers like **Zotero**, **Mendeley**, or **EndNote**. These tools offer more robust control over APA 7th Edition requirements.

> **Disclaimer:** Always double-check your final bibliography against official APA guidelines or your university's specific requirements before submitting your work!
